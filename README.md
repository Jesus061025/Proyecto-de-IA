# Proyecto-de-IA
Clasificador de Recibos Automatiza la gestión financiera diaria. El sistema usa OCR y PLN para extraer monto, fecha y comercio de la foto de un recibo. Un modelo de Machine Learning clasifica el gasto al instante. Esto elimina la transcripción manual, agilizando el registro de gastos en segundos y mejorando su precisión con el uso
